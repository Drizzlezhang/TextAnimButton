## TextAnimButton
文字切换动画Button

### How to use it?
    //属性设置
    default_text_color      //初始文字颜色
    default_text            //初始文字
    done_text_color         //完成文字颜色
    done_text               //完成文字
    text_size               //文字大小
    anim_style              //切换动画方式

#### 也可以通过代码设置
    textAnimButton.setDefaultColor(...);
    textAnimButton.setAnimStyle(TextAnimButton.UP);
